# 社区行为准则

## 我们的承诺

我们致力于为所有参与者提供友善、安全、包容且无骚扰的协作环境，不因年龄、
身体状况、族群、性别认同、经验水平、国籍、外貌、宗教或其他个人特征区别对待。

## 期望行为

- 尊重不同观点和经验，围绕事实与代码讨论。
- 提供具体、建设性的反馈，并主动帮助他人解决问题。
- 对错误负责，接受合理批评并及时改进。
- 保护用户隐私、凭据和未公开的安全信息。

## 不可接受的行为

- 侮辱、威胁、骚扰、歧视或人身攻击。
- 未经允许公开他人的私人信息。
- 故意破坏讨论、代码库或社区协作。
- 任何在专业环境中明显不适当的行为。

## 执行

请通过项目维护者公布的私密联系方式报告问题。维护者可根据影响采取警告、删除
内容、临时限制或永久禁止参与等措施，并会尽合理努力保护报告人的隐私。
