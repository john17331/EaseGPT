# EaseGPT 名称和 Logo 使用规则

## 适用范围

“EaseGPT”名称、EaseGPT Logo、项目视觉标识及其容易造成混淆的变体，统称为
“项目标识”。这些项目标识不包含在 Apache License 2.0 授予的版权或专利许可中。
本规则不表示任何标识已经注册为商标。

## 无需另行许可的使用

- 如实说明软件源自 EaseGPT 项目。
- 在文章、教程、评测或兼容性说明中以文字引用 EaseGPT。
- 保留源代码及发行包中已有的版权、归属和 NOTICE 信息。
- 明确标注为非官方社区分支，并避免让用户误认为其由 EaseGPT 项目维护或背书。

## 需要事先书面许可的使用

- 将 EaseGPT 名称或 Logo 用作公司、产品、服务、域名或商业发行版的主要品牌。
- 使用项目标识暗示官方认证、合作、赞助或背书。
- 修改 Logo 后用于容易与官方项目混淆的发行、宣传或商业活动。
- 在商品、付费服务或广告中突出使用项目标识。

## 分支与衍生版本

衍生版本可以说明“基于 EaseGPT”，但应使用自己清晰可区分的名称和视觉标识。
如果版本与官方项目存在功能或安全差异，应明确说明。不得删除依法或许可证要求
必须保留的归属信息。

## 联系方式

商标授权申请请通过项目仓库维护者提供的公开联系方式提交。在正式发布 GitHub
仓库后，请在此处补充专用联系邮箱或 Discussions 链接。
